# 🏸 バドミントン得点カウンター

スマホ／タブレットのブラウザで動く、バドミントンの得点カウンターです。
HTML/CSS/JS 単一ファイル・外部依存なし・オフライン動作。コート脇のタブレットでそのまま使えます。

大阪女学院 バドミントン部 用に作成。

## 特長

- **15点制（2026年新ルール）／21点制（従来）** を切り替え
- **シングルス／ダブルス** 両対応
- パネルをタップ＝そのサイドがラリーに勝って +1
- サーブ位置（偶数=右／奇数=左）を自動表示。ダブルスはローテーションも計算
- ダブルスは **最初のサーバー／レシーバー** を選択可能
- セット（ゲーム）カウントをスコアボード風に表示
- **ゲームポイント／マッチポイント** を自動表示
- **インターバル60秒**（8点／21点制は11点）・**ゲーム間120秒** の自動タイマー
- 最終ゲームの **エンド交替** を自動アラート（画面の左右入替も任意で選択可）
- 1手戻す（Undo）、ゲーム／マッチ勝敗の自動判定

## 使い方

1. `index.html` をブラウザで開く
2. 得点方式・種目・所属／名前・最初のサーブ側などを設定して「試合開始」
3. ラリーに勝ったサイドのパネルをタップ。間違えたら「1手戻す」

## ルールの根拠

- 公益財団法人 日本バドミントン協会「競技規則」（2024年12月）
- 同協会リリース（2026年5月1日）「3ゲーム×15点制」の国内大会導入方針
- 15点制の細則：競技規則 付録3「代替スコアリングシステム 2（1ゲーム15点・2ゲーム先取）」

## 技術

- 単一の `index.html`（CSS/JS 埋め込み、外部リソースなし）
- ビルド不要・サーバー不要。ファイルを開くだけで動作
